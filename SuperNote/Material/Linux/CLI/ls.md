## Definition
>*`(list)`* is a Linux command used to display files and directories. 
---
## Tags 

#Material #LinuxBasic #CLI

---
### Usage : 
```bash
ls [Option] <FileName>
```
### Options :
| Command | Description                                  |
| ------- | -------------------------------------------- |
| `ls`    | List files and directories                   |
| `-l`    | Long format (permissions, owner, size, date) |
| `-a`    | Include hidden files                         |
| `-la`   | Long format + hidden files                   |
| `-h`    | Human-readable sizes                         |
| `-R`    | Recursive listing (subdirectories)           |
| `-li`   | Check inode                                  |
