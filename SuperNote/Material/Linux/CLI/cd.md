## Definition
>*`(change directory)`* is a Linux Command used to move to destinated directory in file system.

---
## Tags 

#Material #CLI #LinuxBasic 

---
###  Usage : 
```bash
cd [DirName]
```

### Options : 
| DirName   | Effect                   |
| --------- | ------------------------ |
| *`blank`* | Go to home directory     |
| ~         | Go to home directory     |
| ..        | Move to parent directory |
| -         | Switch to prev directory |
| /         | Go to root directory     |

### Others :
- There are no any option for this command.
- For further information please use manual page.
