## Definition
>Hidden files are files or directories whose names start with a dot (`.`). They are hidden by default to reduce clutter and prevent accidental edits.

---
## Tags

#LinuxBasic #Material

---
### Examples:
- `.bashrc`
- `.ssh/`
- `.gitignore`
- `.config/`

### Purpose:
- Store configuration settings
- Keep system/app files out of normal view
- Protect important files from casual modification

### How to View Hidden Files:
```bash
ls -la
```
